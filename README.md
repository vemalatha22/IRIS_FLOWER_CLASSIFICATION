# Iris Flower Classification using Machine Learning

## Overview

This project is a Machine Learning model that classifies Iris flowers into three different species based on their sepal and petal measurements. The model was built using Python and Logistic Regression.

## Dataset

* **Dataset:** Iris Flower Dataset
* **Features:**

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* **Target:**

  * Iris Setosa
  * Iris Versicolor
  * Iris Virginica

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Project Workflow

* Import required libraries
* Load the dataset
* Explore and clean the data
* Perform data visualization
* Split the dataset into training and testing sets
* Train a Logistic Regression model
* Evaluate the model using accuracy, confusion matrix, and classification report
* Predict the species of new flower samples

## Results

The trained model achieved an accuracy of approximately **91%** on the test dataset.
